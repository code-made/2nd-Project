# 🗂️ FileForge - File Organizer


## 🎯 What It Does
Automatically organizes messy folders by sorting files into neat subfolders based on their type!

## ✨ Features
- 📁 Organizes files by extension (.jpg → Images/, .pdf → Documents/, etc.)
- 👀 Preview files before organizing
- 🛡️ Safe - creates new folders, doesn't delete anything
- 📋 Menu-driven interface
- ❓ Built-in help system

## 🚀 Quick Start
1. Save `fileforge.py`
2. Run: `python fileforge.py`
3. Choose option 2 (Organize files)
4. Enter folder path to organize
5. Watch files get organized!

## 📂 File Types Organized
- **Images**: .jpg, .jpeg, .png, .gif → Images/
- **Documents**: .pdf, .txt, .doc, .docx → Documents/
- **Music**: .mp3, .wav, .flac → Music/
- **Videos**: .mp4, .avi, .mkv → Videos/
- **Archives**: .zip, .rar, .7z → Archives/
- **Others**: Everything else → Others/
